# AssemblyScript 学习

## 环境搭建

```
git clone https://github.com/AssemblyScript/assemblyscript.git
cd assemblyscript
npm install
npm link
```

## 目录介绍

```
├── assemblyscript       //assemblyscript 库
├── assembly             //assemblyscript 脚本
├── browser              //浏览器示例
├── nodejs               //nodejs示例
└── package.json         //npm项目描述文件
```
