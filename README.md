# AssemblyScript 学习

## 环境搭建

```
git clone https://github.com/newbieYoung/assemblyscript.git
cd assemblyscript
npm run clean
npm run build
npm install
npm link
```

## 目录介绍

```
├── assemblyscript       //assemblyscript 库
├── assembly             //assemblyscript 脚本
├── build                //asc编译目录
├── browser              //浏览器示例
├── dist                 //webpack打包目录
├── nodejs               //nodejs示例
└── package.json         //npm项目描述文件
```
